# Trading Day Analyzer Pro

这个版本新增了两项核心功能：

1. **K线图交易打点 + 标签可拖拽**
   - 导入 K 线 CSV 后，会在网页中显示蜡烛图。
   - 每笔交易会显示：仓位、点数、Gross、Fee、Net。
   - 标签默认在右侧空白区，可以直接拖拽，虚线引导线会跟着更新。

2. **导入时区确认**
   - 交易 CSV 和 K 线 CSV 都有单独的“时区基准”选择。
   - 图表显示时区也可以单独设置。

## 推荐导入方式

- Webull 订单 CSV：如果你的数据是 EDT，选择 `UTC-04:00`
- TradingView K 线 CSV：如果导出时是 UTC-4，选择 `UTC-04:00`

## 使用步骤

1. 打开 `index.html`
2. 导入交易 CSV，并确认交易 CSV 的时区基准
3. 导入 K 线 CSV，并确认 K 线 CSV 的时区基准
4. 选择日期
5. 需要时拖拽右侧标签，避免文字和 K 线重叠
6. 如有必要，可手动改单笔 fee，页面会自动重算

## 说明

- 默认手续费按：`仓位 × 2 × 每张每边 Fee`
- 默认每张每边 Fee 为 `0.74`
- 仍然支持 fee CSV 覆盖和手动改单笔 fee
- 标签位置会保存在浏览器本地，下次打开还会记住
